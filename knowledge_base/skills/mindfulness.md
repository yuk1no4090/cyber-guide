# 正念练习

## 什么是正念？

正念是一种将注意力有意识地集中在当下此刻的练习，以开放、好奇、不评判的态度观察自己的想法、感受和身体感觉。

## 核心要素

1. **有意识的注意**: 主动将注意力放在当下
2. **当下此刻**: 不沉溺过去，不担忧未来
3. **不评判**: 观察而不判断好坏对错

## 正念呼吸练习（5分钟）

### 步骤
1. 找一个舒适的坐姿
2. 轻轻闭上眼睛
3. 将注意力放在呼吸上
4. 注意气息进出鼻腔的感觉
5. 当思绪飘走时（这很正常！），温柔地将注意力带回呼吸
6. 不需要控制呼吸，只是观察

### 小贴士
- 思绪飘走是正常的，不要责怪自己
- 每次将注意力带回就是一次"正念肌肉"的锻炼

## 身体扫描（10-15分钟）

### 步骤
1. 躺下或坐着，闭上眼睛
2. 从脚趾开始，慢慢向上扫描全身
3. 注意每个部位的感觉：温度、紧张、放松
4. 不需要改变任何东西，只是观察
5. 如果发现紧张的地方，可以想象呼气时将紧张释放

### 顺序参考
脚趾 → 脚掌 → 脚踝 → 小腿 → 膝盖 → 大腿 → 臀部 → 腹部 → 胸部 → 手指 → 手掌 → 前臂 → 上臂 → 肩膀 → 颈部 → 面部 → 头顶

## 五感练习（应急用）

当感到焦虑或脱离当下时，可以用这个"着陆"技术：

### 5-4-3-2-1 技巧
- **5** 样你能**看到**的东西
- **4** 样你能**触摸**到的东西
- **3** 样你能**听到**的声音
- **2** 样你能**闻到**的气味
- **1** 样你能**尝到**的味道

这个练习能快速将注意力带回当下。

## 正念行走

### 步骤
1. 慢慢地走路
2. 注意脚接触地面的感觉
3. 感受重心的转移
4. 注意腿部肌肉的运动
5. 观察周围的环境

### 适用场景
- 午休时间
- 散步时
- 感到烦躁需要活动时

## 正念饮食

### 步骤
1. 在吃东西前，观察食物的颜色、形状
2. 闻一闻气味
3. 第一口放入嘴里，注意味道和口感
4. 慢慢咀嚼，感受味道的变化
5. 感受食物被吞咽的过程

### 好处
- 减少暴饮暴食
- 增加对食物的享受
- 培养当下意识

## 常见误解

❌ 正念是要清空大脑
✅ 正念是观察想法来来去去

❌ 正念只是放松技术
✅ 正念是一种觉察方式

❌ 正念需要很长时间
✅ 即使1分钟也有帮助

## 日常生活中的正念

- **刷牙时**: 注意牙刷的运动和口腔的感觉
- **洗手时**: 感受水的温度和肥皂的触感
- **等待时**: 观察周围环境而不是刷手机
- **倾听时**: 全身心地专注于对方的话

## 开始建议

- 从每天 5 分钟开始
- 选择固定的时间（如起床后或睡前）
- 使用引导音频可能更容易
- 不要追求"完美"的冥想
