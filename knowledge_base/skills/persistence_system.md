# 如何坚持做一件事：700 天背单词的真实经验

## 这不是鸡汤，是一个真实的故事

一个 CS 学生，在墨墨背单词上坚持了超过 700 天，最后雅思考了 7 分。不是天赋异禀，而是找到了一套让自己"不得不坚持"的系统。

## 核心方法论：靠系统，不靠意志力

### 1. 找一个"靶向目标"——给自己设底线

不是"我要每天背单词"这种模糊目标，而是一条**不可违背的底线**：

> 每天必须背完单词再睡觉，无论多晚。

这条规则没有任何弹性。不是"尽量背"，是"背完才能睡"。当规则足够简单、足够绝对，你就不用每天花精力做"今天要不要背"的决定了。

### 2. 碎片化并发——把刷视频的时间换掉

不需要专门腾出一个小时坐在桌前背单词。而是像多线程一样"并发执行"：

- 游戏排队的时候背几个
- 上学路上、坐地铁的时候背
- 吃完饭休息的间隙背
- 把本来刷短视频的碎片时间替换成背单词

关键心态：**不强求一次记住**。第一遍没记住没关系，app 会安排你复习。几轮下来自然就记住了。

### 3. 组队互相监督——社交压力的正向利用

一开始是和同学在墨墨上组队，互相监督。虽然最后只有自己坚持下来了，但**起步阶段的社交承诺很重要**——你不想成为那个掉队的人。

### 4. 等待"飞轮效应"——积累到一定程度会产生加速度

坚持到 400 天左右的时候，参加了一个雅思速成班。这时候发现：自己的单词积累远超其他同学，老师问的复杂单词总能答上来。

这种**成就感**变成了新的驱动力。不再需要靠纪律逼自己，而是发自内心想继续——因为能感受到积累带来的真实回报。

这就是飞轮效应：
```
坚持积累 → 看到回报 → 产生成就感 → 更想继续 → 坚持积累...
```

前面那几百天是最难的，因为还没转起来。但只要撑过去，后面会越来越轻松。

### 5. 巩固优先——不贪新，先把旧的吃透

到了后期，不急着学新单词，而是每天复习之前的。这个策略非常聪明：

- 新学的容易忘，复习的越来越牢
- 基础扎实后，新词学起来更快（因为很多是词根派生的）
- 避免"学了一万个词但每个都只记了一半"的陷阱

## 这套方法可以迁移到别的事情上吗？

核心公式是通用的：

1. **设一条不可违背的底线**（简单、绝对）
2. **利用碎片时间**（不需要大块时间）
3. **不强求完美**（第一遍不会没关系，会重复的）
4. **找到社交监督**（至少起步阶段）
5. **等待飞轮转起来**（前期靠纪律，后期靠成就感）

## 但是——代码和单词不完全一样

背单词是**识别型记忆**，碎片化天然适合。但写代码是**创造型思考**，需要一定的连续时间来进入状态。所以不能完全照搬，需要做区分：

- **适合碎片化的**：看代码片段、刷概念题、读源码、背 API 用法
- **需要整块时间的**：写完整项目、调试 bug、做算法题

最好的策略是两者结合：碎片时间积累知识，整块时间动手实践。
