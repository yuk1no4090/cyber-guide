# 认知重构技巧

## 概述
认知重构是认知行为疗法 (CBT) 的核心技术之一。它帮助我们识别和改变导致负面情绪的不合理思维模式，用更平衡、现实的想法来替代。

## 核心原理
我们的感受不是由事件本身决定的，而是由我们对事件的**解读**决定的。

```
事件 → 想法（解读）→ 情绪/行为
```

通过改变想法，我们可以改变感受。

## 常见的思维陷阱

### 1. 非黑即白（全或无思维）
- **例子**: "如果我不能做到完美，那就是完全失败"
- **替代**: "做得不完美也是有价值的，我可以从中学习"

### 2. 以偏概全
- **例子**: "这次面试失败了，我永远找不到工作"
- **替代**: "这只是一次面试，我还有其他机会"

### 3. 灾难化思维
- **例子**: "如果演讲搞砸了，我的职业生涯就完了"
- **替代**: "即使演讲不顺利，也不会影响全局"

### 4. 读心术
- **例子**: "他没回我消息，一定是讨厌我了"
- **替代**: "他可能只是忙，我不知道真正原因"

### 5. 应该思维
- **例子**: "我应该总是让别人开心"
- **替代**: "我尽力就好，我无法控制他人的感受"

### 6. 情绪化推理
- **例子**: "我感觉自己很没用，所以我就是没用的人"
- **替代**: "感受不等于事实，我只是暂时情绪低落"

## 三栏法练习

这是最常用的认知重构工具：

| 自动化想法 | 思维陷阱 | 更平衡的想法 |
|-----------|---------|------------|
| 我搞砸了，大家都在嘲笑我 | 读心术、灾难化 | 我犯了个错，但大多数人可能没注意到 |
| 我永远做不好这件事 | 以偏概全 | 这次没做好，不代表以后都做不好 |
| 如果我求助，别人会看不起我 | 读心术 | 求助是勇敢的表现，很多人愿意帮忙 |

## 苏格拉底式提问

当你陷入负面思维时，问自己：

1. **证据是什么？**
   - 支持这个想法的证据是什么？
   - 反对这个想法的证据是什么？

2. **有没有其他解释？**
   - 还有别的可能性吗？
   - 如果是朋友遇到这种情况，我会怎么看？

3. **最坏的结果是什么？**
   - 最坏会怎样？
   - 最好会怎样？
   - 最可能会怎样？

4. **这有多重要？**
   - 一年后这件事还重要吗？
   - 五年后呢？

## 实践步骤

### 第一步：觉察
- 注意到自己的情绪变化
- 问自己："我现在在想什么？"

### 第二步：记录
- 写下自动化想法
- 评估情绪强度 (0-100)

### 第三步：检验
- 识别思维陷阱
- 问自己苏格拉底式问题

### 第四步：重构
- 写下更平衡的想法
- 重新评估情绪强度

## 小贴士

- **耐心**: 改变思维模式需要时间和练习
- **不是积极思维**: 目标是更现实，而不是盲目乐观
- **写下来**: 写比想更有效
- **自我关怀**: 对自己温柔一点
