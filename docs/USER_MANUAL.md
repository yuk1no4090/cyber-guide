# User Manual

## 1. What Cyber Guide does

Cyber Guide helps you with:

- study direction confusion
- career planning discussions
- short-term actionable plans
- emotional support during uncertainty

It is not a medical or therapy system.

## 2. Main functions

### 2.0 Account & sidebar

- **Sign in** (email + password, or GitHub): your chat sessions, profile JSON, and 7-day plan rows are tied to your account and reload after refresh.
- **Stay anonymous**: you can chat without an account; starting a **new chat** should not carry over prior server-side session history (only the current thread is remembered for that visit).
- **Collapsible sidebar**: lists past sessions when logged in; pushes the main chat area (not an overlay). Use **New chat** to start a clean thread.
- **Theme**: use the sun/moon control in the header next to login for light/dark mode.

### 2.1 Chat

- type your concern directly
- receive streamed responses and suggestion chips
- when the model uses RAG, open the **evidence** section on an assistant message to see sources and **clickable links** to crawled originals where available
- continue by clicking a suggestion or typing your own follow-up
- **Scenario templates** (relationship / stress presets) apply a dedicated prompt strategy for that turn

### 2.2 Profile analysis

- switch to profile mode
- choose self-analysis or "analyze someone else"
- fill the structured profile form (school, goals, internships, etc.)
- you may see **similar background cases** with short explanations and source links
- complete the flow and generate a **report**; you can also get a **dual-path** (e.g. study vs job) style answer where implemented

### 2.3 7-day plan

- ask for a 7-day plan
- track each day as:
  - todo
  - done
  - skipped
- regenerate one day when current task is not realistic

### 2.4 Feedback

- rate conversation quality
- optional text feedback improves future quality

## 3. Privacy and safety

- data logging is opt-in
- sensitive text is redacted before persistence
- crisis signals trigger immediate safety response

If you are in urgent danger, contact local emergency services immediately.

## 4. Tips for better results

- provide context:
  - current stage
  - concrete blockers
  - what you already tried
- ask for:
  - "拆成 3 步"
  - "先做今天最小动作"
  - "给我两个可选方向"

## 5. Known limits

- model output can be imperfect
- crawler content may lag behind real-time updates
- planning suggestions are advisory, not guaranteed outcomes
