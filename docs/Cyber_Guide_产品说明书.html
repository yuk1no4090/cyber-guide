1. 产品一句话介绍

Cyber Guide 是一款面向学业与职业选择的 AI 规划助手。用户输入自己的学校、成绩、排名、实习、科研、竞赛与目标方向后，系统会检索相似背景案例，结合 AI 生成个性化建议，并提供原文链接与可执行计划。

区别于普通聊天机器人，Cyber Guide 的重点不是“泛泛给建议”，而是“找到和用户类似的人走过什么路”。

2. 目标用户与典型痛点

计算机相关学生
  • 不知道自己的学校层次和成绩能匹配哪些学校。
  • 保研、考研、留学、就业几条路之间难以取舍。
  • 普通一本、双非、小众院校缺少同校经验帖。
早期职场人与转码人群
  • 不清楚自己的项目、实习、学历对求职竞争力的影响。
  • 需要面经、薪资、岗位选择和路线建议。
  • 想知道“相似背景的人最后去了哪里”。

3. 产品核心特色

相似背景检索

系统会识别用户学校、学校层次、GPA、排名、实习、科研、竞赛和目标方向，并从案例库中找出背景接近的真实案例。

原文可验证

AI 回答会尽量引用真实经验帖，并附上可点击原文链接，减少“AI 编故事”的不确定性。

多路径规划

覆盖保研、考研、留学、就业、GAP、转码等方向；用户意向不明确时，可同时给出升学与就业两条路径。

7 天行动计划

用户不仅能得到结论，还能生成 7 天微行动计划，将规划拆成可执行任务。

结构化画像

支持录入学校、GPA、排名、实习、科研、竞赛和目标方向，让建议更贴合个人情况。

登录后持久化

登录用户可保存聊天历史、用户画像、行动计划和会话数据；匿名用户则保持轻量使用体验。

5. 相似背景检索能力

这是 Cyber Guide 的核心差异化能力。系统不只根据关键词搜索，而是将用户背景和案例背景拆成结构化要素，再计算相似度。

检索逻辑示例

当用户说：“我是双非计算机大三，GPA 3.5，排名前 10%，想保研去 985”，系统会优先检索：

  • 学校层次相近：双非、普通一本、双非强校。
  • 成绩接近：GPA 3.2 - 3.8、排名前 5% - 15%。
  • 目标一致：保研/推免/夏令营案例。
  • 经历相似：有科研、实习或竞赛加分。
  • 输出可验证：附原文链接与案例摘要。

6. 数据与知识来源

Cyber Guide 通过爬虫与结构化抽取构建案例库，主要覆盖知乎、掘金、V2EX、CSDN、考研/保研相关站点等来源。

数据类型内容用途
经验帖正文考研上岸、保研夏令营、留学申请、秋招面经、实习转正等。作为 RAG 检索证据。
结构化案例学校、层次、GPA、排名、去向、目标学校、实习/科研/竞赛。用于相似度排序。
学校映射库国内高校层次与排名、海外 QS 学校及别名。解决小众学校匹配问题。
知识库文档规划方法、路线建议、专业技能建议。补充通用指导。

7. 技术架构概览

用户输入背景
画像提取
RAG 检索案例
AI 生成建议
链接与计划输出

前端

Next.js + React + Tailwind，包含聊天界面、侧边栏、登录注册、画像表单、相似案例卡片、证据面板与行动计划组件。

后端

Spring Boot + PostgreSQL + Redis，使用责任链 Pipeline 处理消息，包括脱敏、风控、RAG 检索、AI 调用和响应解析。

爬虫与数据加工

Python 爬虫负责抓取文章；补全文脚本拉取完整正文；结构化抽取脚本将经验帖转为可匹配字段。

8. 用户使用流程

  1. 用户进入页面，输入自己的背景或打开画像表单。
  2. 系统识别学校层次、GPA、排名、经历与目标方向。
  3. RAG 从真实案例中检索相似背景用户的路径。
  4. AI 生成简洁建议，并给出案例引用和原文链接。
  5. 用户可继续追问，也可生成 7 天行动计划。
  6. 登录用户可保存历史会话、画像和计划。

10. 产品优势总结

  • 更接近真实决策：不只给泛建议，而是结合相似背景案例。
  • 对小众学校更友好：通过学校层次、排名区间和群体标签匹配，不依赖同校案例。
  • 可验证：重要建议尽量附上原文链接和证据来源。
  • 可执行：支持将方向建议拆成短周期行动计划。
  • 可持续进化：爬虫、反馈、结构化抽取和评估脚本让案例库持续增长。
